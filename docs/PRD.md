## 🎯 제품 개요

### 제품 설명

**Scientific Calculator**는 모바일 우선으로 설계된 현대적인 과학 계산기 웹 애플리케이션입니다. 직관적인 인터페이스와 강력한 계산 기능을 제공하여 학생, 엔지니어, 연구원들이 빠르고 정확하게 수학 연산을 수행할 수 있도록 돕습니다.

### 핵심 가치 제안

- ✅ **모바일 최적화**: 스마트폰에서 완벽하게 작동하는 반응형 디자인
- ✅ **직관적인 UI**: 물리적 계산기와 유사한 친숙한 인터페이스
- ✅ **과학 함수**: 삼각함수, 로그, 제곱근 등 고급 수학 함수 지원
- ✅ **다크 모드**: 눈의 피로를 줄이는 세련된 다크 테마
- ✅ **계산 기록**: 이전 계산 결과 저장 및 재사용
- ✅ **오프라인 작동**: PWA로 인터넷 없이도 사용 가능

## 🎨 목표 및 비전

### 단기 목표 (MVP - 1개월)

1. **기본 계산기 기능 완성**

   - 기록에서 재사용

3. **모바일 최적화**
   - 반응형 디자인
   - 터치 최적화
   - 햅틱 피드백 (지원 기기)

- 단위 변환기 통합
- 수학/물리 상수 라이브러리
- 그래프 플로팅 기능
- 방정식 풀이 기능
- 다국어 지원

## ✨ 핵심 기능

### MVP 기능 (우선순위 P0)

#### 1. 기본 계산 기능
- **사칙연산**: +, -, ×, ÷
- **백분율**: %
- **부호 변경**: +/-
- **소수점**: .
- **지우기**: AC (전체 삭제), ← (한 글자 삭제)

#### 2. 과학 함수
- **삼각함수**: sin, cos, tan
- **로그**: ln (자연로그), log (상용로그)
- **제곱/제곱근**: x², √
- **각도 모드**: deg (도), rad (라디안)

#### 3. 계산 기록
- **자동 저장**: 계산 완료 시 자동으로 기록
- **기록 보기**: 최근 50개 계산 내역
- **재사용**: 기록 탭하여 표현식 불러오기
- **삭제**: 개별 또는 전체 삭제

#### 4. 설정
- **테마**: 라이트/다크 모드
- **햅틱 피드백**: 버튼 터치 시 진동 (선택)
- **정밀도**: 소수점 자리수 설정

## 📋 기능 요구사항

### 1. 계산 엔진

#### 1.1 기본 연산
- **입력**: 숫자 버튼 (0-9), 연산자 (+, -, ×, ÷)
- **처리**: 표현식 평가 (mathjs 라이브러리 사용)
- **출력**: 결과 표시 (최대 10자리 정밀도)

**예시**:
```
입력: 2 + 2 =
출력: 4

입력: 10 ÷ 3 =
출력: 3.333333333
```

#### 1.2 과학 함수
- **삼각함수**: sin, cos, tan
  - 각도 모드에 따라 도/라디안 변환
  - 예: sin(30°) = 0.5, sin(π/6 rad) = 0.5

- **로그 함수**: ln, log
  - ln: 자연로그 (밑 e)
  - log: 상용로그 (밑 10)

- **제곱/제곱근**: x², √
  - x²: 현재 값의 제곱
  - √: 현재 값의 제곱근

#### 1.3 에러 처리
- **0으로 나누기**: "Error: Division by zero"
- **잘못된 입력**: "Error: Invalid expression"
- **범위 초과**: "Error: Number too large"

### 2. 계산 기록

#### 2.1 저장
- **자동 저장**: = 버튼 누를 때마다 저장
- **저장 형식**: `표현식 = 결과`
- **저장 위치**: LocalStorage
- **최대 개수**: 50개 (오래된 것부터 삭제)

**예시**:
```
sin(30) + 5 = 5.5
10 ÷ 3 = 3.333333333
√16 = 4
```

#### 2.2 조회
- **정렬**: 최신순 (위에서부터)
- **표시**: 표현식과 결과
- **스크롤**: 무한 스크롤

#### 2.3 재사용
- **탭**: 기록 항목 탭하면 표현식 불러오기
- **편집**: 불러온 표현식 수정 가능

#### 2.4 삭제
- **개별 삭제**: 스와이프하여 삭제
- **전체 삭제**: "모두 삭제" 버튼

### 3. 설정

#### 3.1 테마
- **옵션**: 라이트, 다크, 시스템 설정 따르기
- **저장**: LocalStorage에 저장
- **적용**: 즉시 반영

#### 3.2 각도 모드
- **옵션**: 도(deg), 라디안(rad)
- **표시**: 버튼에 현재 모드 표시
- **전환**: 버튼 탭하여 토글

#### 3.3 햅틱 피드백
- **옵션**: 켜기/끄기
- **적용**: 버튼 탭 시 짧은 진동
- **지원**: 지원 기기만 (iOS, Android)

## 🚀 향후 확장 계획

### Phase 2 (2-3개월)

#### 단위 변환기
- **길이**: m, km, cm, mm, in, ft, yd, mi
- **무게**: kg, g, mg, lb, oz
- **온도**: °C, °F, K
- **속도**: m/s, km/h, mph
- **면적**: m², km², ft², acre
- **부피**: L, mL, gal, cup

#### 상수 라이브러리
- **수학 상수**: π, e, φ (황금비)
- **물리 상수**: c (광속), G (중력상수), h (플랑크 상수)
- **화학 상수**: NA (아보가드로 수), R (기체상수)

### Phase 3 (4-6개월)

#### 고급 기능
- **그래프 플로팅**: 함수 그래프 시각화
- **방정식 풀이**: 1차, 2차 방정식 해 구하기
- **행렬 계산**: 덧셈, 곱셈, 역행렬
- **통계**: 평균, 표준편차, 분산

#### 협업 기능
- **계산 공유**: URL로 계산 결과 공유
- **내보내기**: PDF, 이미지로 저장
- **클라우드 동기화**: 여러 기기 간 기록 동기화
