# 공학용 계산기 웹앱 - 제품 요구사항 명세서 (PRD)
**Product Requirements Document**

---

## 1. 요약

### 1.1 제품 비전
Engineering Calc는 학생, 엔지니어, 전문가들에게 여러 공학 분야의 포괄적인 공식 라이브러리에 즉시 접근할 수 있는 현대적인 웹 기반 공학용 계산기 애플리케이션입니다. 이 애플리케이션은 공식 참조, 대화형 계산, 프리미엄 사용자 경험을 하나의 직관적인 인터페이스로 결합합니다.

### 1.2 타겟 사용자
- **주요**: 공학 전공 학생 (학부 및 대학원)
- **부차적**: 전문 엔지니어, 연구원, 기술 전문가
- **3차**: 교육자 및 학술 강사

### 1.3 핵심 가치 제안
- **포괄적인 공식 라이브러리**: 물리학, 수학, 화학, 전기공학, 토목공학 분야의 24개 이상의 공식 제공
- **대화형 계산**: 단위 변환을 지원하는 실시간 계산
- **현대적인 UX**: 부드러운 애니메이션과 반응형 디자인을 갖춘 프리미엄 다크 테마 인터페이스
- **체계적인 접근**: 검색, 즐겨찾기, 히스토리를 포함한 카테고리 기반 네비게이션

---

## 2. 제품 개요

### 2.1 제품 유형
반응형 디자인을 갖춘 단일 페이지 웹 애플리케이션 (SPA)

### 2.2 주요 기능
1. 카테고리 필터링이 가능한 공식 라이브러리 브라우저
2. 단위 변환 기능을 갖춘 대화형 공식 계산기
3. 모든 공식에 대한 검색 기능
4. 사용자 개인화 (즐겨찾기, 히스토리)
5. 공식 문서 및 관련 참조
6. 결과 시각화 및 내보내기 옵션

#### 타이포그래피

- Border radius: 0.25rem (기본), 0.5rem (lg), 0.75rem (xl), 1rem (2xl)

### 3.2 레이아웃 구조

애플리케이션은 **3-패널 레이아웃**을 사용합니다:

```
┌─────────────────────────────────────────────────────────┐
│                    상단 네비게이션 바                      │
├──────────┬─────────────────┬───────────────────────────┤
│          │                 │                           │
│ 카테고리  │   공식 목록      │    상세 작업 공간          │
│ 사이드바  │   (중간)        │      (메인 콘텐츠)         │
│  (왼쪽)  │                 │                           │
│          │                 │                           │
└──────────┴─────────────────┴───────────────────────────┘
```

#### 상단 네비게이션 (헤더)
- **왼쪽**: 앱 로고 + 브랜딩 ("Engineering Calc")
- **중앙**: 메인 네비게이션 (대시보드, 라이브러리, 히스토리, 설정)
- **오른쪽**: 알림 아이콘 + 사용자 프로필 아바타
- **높이**: 고정, ~60px
- **배경**: 다크 (`#111a22`) 하단 테두리 포함

#### 왼쪽 사이드바 (카테고리 네비게이션)
- **너비**: 256px (모바일에서 숨김)
- **섹션**:
  - **학문 카테고리**: 전체 카테고리, 수학, 물리학, 화학, 전기공학, 토목공학
  - **라이브러리 필터**: 즐겨찾기, 최근 항목
- **기능**:
  - Material Symbols를 사용한 아이콘 기반 네비게이션
  - 활성 상태 표시 (왼쪽 테두리 강조, 하이라이트 배경)
  - 색상 전환을 포함한 호버 효과

#### 중간 패널 (공식 목록)
- **너비**: 320px (모바일: 전체 너비), 384px (lg)
- **구성 요소**:
  - 아이콘이 있는 검색 바
  - 공식 개수 + 정렬 버튼
  - 스크롤 가능한 공식 카드
- **공식 카드 디자인**:
  - 제목 (공식 이름)
  - 공식 미리보기 (중앙 정렬, 세리프 이탤릭)
  - 카테고리 태그
  - 활성 상태: 왼쪽 테두리 강조, 하이라이트 배경, 그림자
  - 호버 상태: 배경 변경, 테두리 표시

#### 오른쪽 패널 (상세 작업 공간)
- **너비**: 유연함 (남은 공간)
- **섹션**:
  1. **브레드크럼 네비게이션**: 라이브러리 > 카테고리 > 하위 카테고리
  2. **액션 버튼**: 즐겨찾기, 공유, 도움말
  3. **공식 제목 및 설명**: 설명 텍스트가 포함된 큰 제목
  4. **히어로 방정식 표시**: 그라디언트 배경이 있는 크고 중앙 정렬된 공식
  5. **계산기 섹션**: 
     - 입력 변수 (왼쪽, 2/3 너비)
     - 결과 패널 (오른쪽, 1/3 너비)
  6. **추가 정보**: 공학 노트, 관련 공식

## 4. 기능 요구사항

### 4.1 공식 라이브러리 관리

#### FR-1: 카테고리 네비게이션
- **설명**: 사용자가 공학 분야별로 공식을 탐색할 수 있음
- **카테고리**:
  - 전체 카테고리 (모든 공식 표시)
  - 수학
  - 물리학
  - 화학
  - 전기공학
  - 토목공학
- **동작**: 카테고리를 클릭하면 해당 분야의 공식만 표시하도록 공식 목록 필터링

#### FR-2: 공식 검색
- **설명**: 공식 이름 및 카테고리에 대한 실시간 검색
- **기능**:
  - 입력하는 대로 검색 기능
  - 대소문자 구분 없는 매칭
  - 일치하는 결과 하이라이트
- **입력**: 검색 아이콘이 있는 텍스트 필드
- **출력**: 필터링된 공식 목록

#### FR-3: 공식 정렬
- **설명**: 사용자가 다양한 기준으로 공식을 정렬할 수 있음
- **정렬 옵션**:
  - 알파벳순 (A-Z)
  - 최근 사용
  - 가장 자주 사용
  - 카테고리

#### FR-4: 공식 상세 정보 표시
- **설명**: 공식 카드를 클릭하면 오른쪽 패널에 전체 세부 정보 표시
- **표시 정보**:
  - 공식 이름 및 설명
  - 수학 방정식 (큰 표시)
  - 변수 정의
  - 단위 정보
  - 공학 노트
  - 관련 공식
  - 대화형 계산기

### 4.2 계산기 기능

#### FR-5: 변수 입력
- **설명**: 사용자가 공식의 각 변수에 대한 값을 입력할 수 있음
- **기능**:
  - 검증 기능이 있는 숫자 입력 필드
  - 각 변수에 대한 단위 선택 드롭다운
  - 변수 유형 표시 (스칼라/벡터)
  - 변수 설명을 위한 정보 버튼
  - 입력 검증 (숫자만, 범위 확인)

#### FR-6: 단위 변환
- **설명**: 입력 및 출력에 대한 자동 단위 변환
- **지원 단위 카테고리**:
  - 질량: kg, g, lb
  - 가속도: m/s², ft/s², g
  - 힘: N, lbf, kN
  - 에너지: J, kJ, cal, BTU
  - (공식 요구사항에 따라 확장 가능)

#### FR-7: 계산 실행
- **설명**: 입력이 제공되면 실시간 계산
- **동작**:
  - "계산" 버튼이 계산 트리거
  - 전용 패널에 결과 표시
  - 잘못된 입력에 대한 오류 처리
  - 정밀도: 소수점 2-4자리

#### FR-8: 결과 표시
- **설명**: 계산 결과의 명확한 표시
- **기능**:
  - 큰 숫자 표시
  - 단위 표시
  - 결과 설명
  - 클립보드에 복사 버튼
  - 그래프 플롯 옵션 (해당 공식의 경우)

### 4.3 사용자 개인화

#### FR-9: 즐겨찾기
- **설명**: 사용자가 자주 사용하는 공식을 저장할 수 있음
- **기능**:
  - 즐겨찾기 추가/제거를 위한 별 아이콘
  - 사이드바의 즐겨찾기 필터
  - 영구 저장 (localStorage 또는 백엔드)

#### FR-10: 히스토리
- **설명**: 최근 사용한 공식 및 계산 추적
- **기능**:
  - 공식 사용의 자동 로깅
  - 사이드바의 최근 항목 필터
  - 마지막 사용 타임스탬프
  - 히스토리 지우기 옵션

#### FR-11: 설정
- **설명**: 사용자 기본 설정 및 구성
- **옵션**:
  - 기본 단위 시스템 (SI/Imperial)
  - 소수점 정밀도
  - 테마 기본 설정 (다크/라이트)
  - 언어 선택

### 4.4 추가 기능

#### FR-12: 공식 공유
- **설명**: 공식 또는 계산 결과 공유
- **방법**:
  - 공식 링크 복사
  - 계산을 PDF로 내보내기
  - 이메일/소셜 미디어를 통한 공유

#### FR-13: 문서 접근
- **설명**: 각 공식에 대한 도움말 및 문서
- **내용**:
  - 유도 설명
  - 사용 사례 및 예제
  - 제한 사항 및 가정
  - 참조 및 출처

#### FR-14: 관련 공식
- **설명**: 관련 공식으로의 빠른 네비게이션
- **표시**: 관련 공식으로 연결되는 클릭 가능한 태그
- **예시**: 
  - 뉴턴의 제2법칙 → 무게 (W=mg), 충격량 (J=FΔt)

## 5. 기술 요구사항

### 5.1 기술 스택

#### 프론트엔드
- **프레임워크**: React.js 또는 Vue.js (SPA)

- **아이콘**: Material Symbols (Google)

  - Space Grotesk (display)

- **API**: RESTful API 또는 GraphQL
- **데이터베이스**: PostgreSQL 또는 MongoDB (사용자 데이터, 히스토리용)

- **프론트엔드**: Vercel, Netlify, 또는 GitHub Pages

## 7. 사용자 스토리

### 에픽 1: 공식 발견
- **US-1**: 학생으로서, 나는 카테고리별로 공식을 탐색하여 내 과제에 관련된 방정식을 찾고 싶다
- **US-2**: 엔지니어로서, 나는 이름으로 공식을 검색하여 필요한 것을 빠르게 찾고 싶다
- **US-3**: 사용자로서, 나는 목록에서 공식 미리보기를 보고 올바른 공식을 한눈에 식별하고 싶다

### 에픽 2: 계산
- **US-4**: 사용자로서, 나는 값을 입력하고 즉시 결과를 얻어 내 계산을 확인하고 싶다
- **US-5**: 사용자로서, 나는 단위 간 변환을 하여 다른 측정 시스템으로 작업하고 싶다
- **US-6**: 사용자로서, 나는 결과를 클립보드에 복사하여 다른 애플리케이션에서 사용하고 싶다

### 에픽 3: 개인화
- **US-7**: 자주 사용하는 사용자로서, 나는 즐겨찾는 공식을 저장하여 빠르게 접근하고 싶다
- **US-8**: 사용자로서, 나는 내 계산 히스토리를 보고 이전 작업을 참조하고 싶다

## 10. 디자인 사양

### 10.1 컴포넌트 라이브러리

- **배경**: Surface border (`#233648`)
- **포커스 상태**: Primary 테두리와 링
- **검증**: 오류의 경우 빨간색 테두리, 성공의 경우 녹색

#### 카드
- **기본**: 테두리가 있는 다크 서페이스
- **활성**: Primary 왼쪽 테두리, 하이라이트 배경, 그림자
- **호버**: 배경 변경, 테두리 표시

#### 네비게이션
- **활성**: Primary 색상 텍스트, 하단 테두리 (수평 네비게이션)
- **활성**: 왼쪽 테두리 강조, 하이라이트 배경 (수직 네비게이션)
- **호버**: 텍스트 색상이 primary로 변경
